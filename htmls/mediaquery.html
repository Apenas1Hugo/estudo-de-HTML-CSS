Exemplo de Media Queries

As media queries são uma funcionalidade do CSS que permite aplicar estilos diferentes para diferentes tamanhos de tela. Isso é útil para criar layouts responsivos que se adaptam a diferentes dispositivos, como desktops, tablets e smartphones.

Este é um box responsivo.

Redimensione a janela do navegador para ver as mudanças de estilo no box acima. Dependendo da largura da tela, a cor de fundo e o tamanho do texto do box mudarão.
