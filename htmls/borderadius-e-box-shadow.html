Exemplo de Border Radius e Box Shadow

A propriedade border-radius permite arredondar os cantos de uma caixa.

border-radius: 10px;

border-radius: 50px;

border-radius: 50%; (forma um círculo)

A propriedade box-shadow adiciona uma sombra à caixa.

box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.2);

box-shadow: 10px 10px 20px rgba(0, 0, 0, 0.4);

box-shadow: 5px 5px 15px rgba(255, 0, 0, 0.6); (sombra colorida)
